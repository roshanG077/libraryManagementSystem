Rules & Instructions

Return On Time

All books are issued for a maximum of 14 days. A daily penalty is automatically applied for late returns according to the active library rate.

Handle With Care

Keep books in excellent condition. Do not fold pages, highlight text, or write inside. Damaged books will require full replacement cost.

Issue Limits

Standard members can issue up to 3 books simultaneously. Please return a book before attempting to issue a new one to keep the system balanced.
